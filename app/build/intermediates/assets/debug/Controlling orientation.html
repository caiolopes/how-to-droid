Controlling screen orientation

To force the screen orientation of an activity, either use the setRequestedOrientation(int) method, e.g., 
public void onCreate(Bundle savedInstanceState) {
  super.onCreate(savedInstanceState);
  setRequestedOrientation(ActivityInfo.SCREEN_ORIENTATION_PORTRAIT);
  ...
}
or specify the android:screenOrientation attribute of an activity in the AndroidManifest.xml file, e.g., 
<activity
  android:name=".MainActivity"
  android:label="@string/app_name"
  android:screenOrientation="portrait">
  ...
</activity>
