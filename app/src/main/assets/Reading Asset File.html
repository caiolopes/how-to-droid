Reading Asset File

Using the AssetManager class, you can access any files contained in the assets directory or its subdirectories. An instance of the AssetManager class can be obtained by calling the getAssets() method on a context object such as an activity. 

AssetManager assetManager = getAssets();
BufferedReader reader = new BufferedReader(
  new InputStreamReader(assetManager.open("myfile.txt")));
String line = null;
while ((line = reader.readLine()) != null) {
  ... process line
}
The URL syntax for assets is file:///android_asset/, e.g., file:///android_asset/myfile.txt; note the use of three slashes.
