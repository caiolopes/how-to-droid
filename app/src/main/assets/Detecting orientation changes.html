Detecting screen orientation changes

To detect the layout changes from portrait to landscape (or vice versa), override the onConfigurationChanged(Configuration) method of the Activity class. 

public void onConfigurationChanged(Configuration newConfig) {
  super.onConfigurationChanged(newConfig);
  if (newConfig.orientation == Configuration.ORIENTATION_LANDSCAPE) {
     ... landscape
  } else if (newConfig.orientation == Configuration.ORIENTATION_PORTRAIT){
     ... portrait
  }
}
One also have to edit the appropriate element in the manifest file to include the android:configChanges. 
<activity 
  android:name=".MainActivity"
  android:configChanges="orientation|keyboardHidden"
  android:label="@string/app_name">
  ...
</activity>
